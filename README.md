## Generate_Quotes

### 名言を生成する

http://quotes.toscrape.com から名言をスクレイピングしてデータベースに保存しておいた.  
スクレイピングにはgo langを使用した.  

そのデータを使って、seq2seqモデルを用い、新たな名言を作成するデスクトップアプリを作った.  
(python app.py で実行できる)
